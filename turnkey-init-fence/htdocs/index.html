You need to initialize this appliance first before you can use it. To do that you'll need to log in as root. The initialization program will start automatically. You can log in using the or using a native SSH client (e.g., Putty on Windows).

If you haven't setup SSH key authentication, you'll need to get the random root password from the EC2 console. It's printed at the bottom so please scroll all the way down.

For further information regarding this appliance, please consult the product page on the TurnKey Linux website.
